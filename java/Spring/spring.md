## spring
### 1 spring优化java开发
    1、基于pojo的轻量级及最小侵入性编程
    2、通过依赖注入和面向接口实现松耦合
    3、基于切面和惯例进行声明式编程
    4、通过切面和模板见识样板式代码
### 2 Spring框架的核心功能有两个
    1、Spring容器作为超级大工厂，负责创建、管理所有的Java对象，这些Java对象被称为Bean
    2、Spring容器管理容器中Bean之间的依赖关系，Spring使用一种被称为“依赖注入”的方式来管理Bean之间的依赖关系

### Spring Mvc
  1. DispatcherServlet
  2. doService
  3. request里面添加一些参数
  4. doDispatch
